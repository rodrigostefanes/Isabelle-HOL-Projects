# Isabelle/HOL Projects

This repository contains a collection of my exploratory formalizations and exercises using the **Isabelle/HOL** interactive theorem prover. 

## Contents

* **GroupTheory:** Basic formalizations of groups and algebraic structures.
* **Combinatorics:** Exploratory implementations and proofs related to ranking and unranking algorithms.
* **SeparationLogic:** Toy example using AutoCorrode
* **Miscellaneous:** Proofs and exercises from the book Concrete Semantics by Tobias Nipkow and Gerwin Klein
